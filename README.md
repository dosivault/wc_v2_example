## DOSI Vault Mobile (iOS, Android)과 연동하는 두가지 방법

1. QR Code (Desktop에서 QR Code를 보여주고 Mobile에서 촬영)
2. Firebase Dynamic Link (Mobile browser에서 redirect를 통해 연결)
